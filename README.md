# InsightFlow

InsightFlow is a data analysis agent platform.

## Current Stage

FastAPI backend with auth, dataset upload, data catalog, analysis sessions, and a minimal Agent API. React frontend foundation for the demo workflow.

## Tech Stack

- FastAPI
- PostgreSQL
- SQLAlchemy
- Alembic
- React
- Vite
- TypeScript
- Tailwind CSS
- LangGraph
- Pandas

## Frontend Dev

```powershell
cd frontend
npm install
npm run dev
```

Frontend URL:

```text
http://localhost:5173
```

Backend URL:

```text
http://localhost:8000
```
